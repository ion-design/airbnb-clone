## About
Trying to clone the complete UI for AirBnb for fun and practice. 

Right now, only the Header is done which in itself is humongous. Will add more UI elements with time.

## ✨ New Feature: Image-Heavy Property Cards

The property card component has been redesigned with an image-heavy, minimal overlay design:

- **Multi-image support** with thumbnail navigation
- **Minimal overlays** that don't obstruct the images
- **Interactive favorite button** with heart icon
- **Fully keyboard accessible** navigation
- **Responsive design** that works on all screen sizes

### View the Design

- **Main page** (`/`) - See the cards integrated with the main property grid
- **Showcase page** (`/showcase`) - View a dedicated demo with all features

📖 See [PROPERTY_CARD_INTEGRATION.md](./PROPERTY_CARD_INTEGRATION.md) for complete documentation.


## Run Locally
Open a terminal and copy-paste the following commands -
```
git clone https://github.com/JatinBumbra/airbnb-clone.git
```
```
cd airbnb-clone
```
```
npm install
```
```
npm run dev
```
